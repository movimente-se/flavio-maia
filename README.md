## Sobre o livro

Muitas pessoas sentem que a vida está parada, como se algo estivesse bloqueando seus sonhos, projetos e decisões.

**MOVIMENTE-SE** é um convite direto para romper a inércia, despertar a consciência e dar passos práticos rumo a uma vida com mais propósito, direção e ação.

Se você sente que nasceu para avançar e não quer viver apenas esperando as coisas acontecerem, este livro é para você.
